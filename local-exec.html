Welcome to Terraform Local-exec Concepts!

This page describe about Terraform Local-exec provisioner allows us to invoke a local executable after the resource is created.One of the most used approach of local-exec is to run ansible-playbooks on the created server after the resource is created..

For online documentation and support please refer to https://developer.hashicorp.com/terraform/language/resources/provisioners/local-exec/.
